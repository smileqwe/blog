命令说明

dev:electron: 使用 ./app/tsconfig.json 配置编译 ./app 目录，编译完成后，等待 react 启动完成，然后使用编译生成的 ./build/app/main.js 入口文件启动 electron 程序
dev:react: 启动 react 本地开发模式
